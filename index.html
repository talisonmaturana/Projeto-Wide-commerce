rFinder

Este projeto, desenvolvi na faculdade para a "Fatec de portas abertas", que é um evento que ocorre anualmente na Fatec. Ele tem o objetivo de ajudar os clientes à localizarem produtos dentro de lojas de auto-atendimento, onde quase não há vendedores e são enormes.

Controlador de Acesso

Desenvolvi para o meu projeto integrador na faculdade, onde tinha que usar o "Arduino" como hardware para deixar "físico" esse controlador. Foi usado também o sensor "RFID" para fazer a leitura dos dados.

Controlador de Luz Mobile

Esse projeto, desenvolvi quando estava fazendo alguns teste usando o arduino. Foi ai que conheci o "Node JS", usando HTML, CSS, JavaScript, Firebase e o Arduino, fiz uma página web para ligar e desligar um LED.

Projeto Bichinhos

Neste, foi desenvolvido a estrutura de um site, quando eu estava aprendendo a usar o "Bootstrap Framework". Este site tinha a finalidade de expor uma galeria de fotos de animais para adoção, onde foi usado o sistema de "Grids" para separar a página igualmente.

T.I. Series

Este projeto foi desenvolvido no ultimo semestre no meu curso de Técnico em Informática, onde era necessário fazer o controle de Séries de TV. Lá você pode cadastrar suas series, selecionar as temporadas e epsódios, etc.

T.I Games

Foi o meu TCC do curso de Técnico em Informática, foi usado o "Construct2" como ferramenta de desenvolvimento. Ele é um jogo, onde pode-se aprender um pouco como funciona a programação orientada à objetos, o que muitas pessoas tem dificuldae em entender e/ou explicar este paradigma.

ATM - Caixa Eletrônico

Foi desenvolvido em sala de aula, tentando imitar o funcionamento de caixas eletrônicos reais, foi usando a linguagem C# e foi orientado à objetos, nele fizemos alguns formulários como se fossem as telas dos caixas.

Cadastro de Clientes

Foi desenvolvido um aplicativo para cadastro de clientes, um protótipo, usando a linguagem Java para a plataforma Android. Também foi usando o SQLite como banco de dados para o armazenamento do cadastro.

Calculadora

Neste foi criada uma calculadora simples, mas o diferencial dela, foi que ela era uma aplicação híbrida, ou seja, multi-plataforma. Foi usado o "Apache Cordova" para o desenvolvimento usando a linguagem HTML e CSS para estrutura e estilo o aplicativo e JavaScript para os calculos.
